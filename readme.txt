Projet:
Optimisation d'un parcours de graphe dans un polygone créé depuis le site data.shom.fr

Language: 
	Java (en développement)
	C (en attente)
	C# (en attente)

Objectif obligatoire:
	Scanner le fichier KML (export du data.shom.fr)
	Créer le graphe depuis le point de départ du drone
	Créer au moins une solution
	Exporter cette solution au format pdf

Objectif secondaire: 
	A voir : readme pour chaque langage

Contrainte:
	Le drone a une porté de vision
	Le drone a potentiellement une jauge de carburant
	Il peut exister des polygones non autorisé ou le drone ne peut pas passer

Données:
	Coordonnées du Polygone
	Point de départ du Drone
	Nombre de déplacement maximale du drone
	Distance de vision
